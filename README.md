# tob-less

<br />

## 介绍

一个主题的 **uniapp** 样式库

<br />
<br />
<br />

## features

1. 原子的
   - 开发效率直接翻一番
2. 主题驱动
   - 自定义主题，满足个性化需求 
3. 支持Mixins
   - 语义化的模板，更清晰的结构 

<br />
<br />
<br />

## 启发 😀

该库受以下技术启发

- [Less](https://less.bootcss.com/)
- [Css 变量](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Using_CSS_custom_properties)
- [Windicss](https://cn.windicss.org/)
- [Tailwindcss](https://www.tailwindcss.cn/)
